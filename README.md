<h1>Grass Is Greener App</h1> 

<h4>This project is a console app that calls data such as country name, local country name, capital, language, population, surface area, population density pertaining to a country that the user specifies from a REST API.</h4>

<h4>To run this app, clone the repository into your IDE and simply run the main.py file. You will be able to interact with the app via your console.</h4>
